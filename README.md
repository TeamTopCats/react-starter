## Installation

#### yarn

```bash
git clone -b master git@github.com:lnked/react-starter.git
```

OR

```bash
git init
git remote add origin git@github.com:lnked/react-starter.git
git fetch
```

Then run command

```bash
yarn install
```

## Run tasks

```bash
yarn install
```

Start dev-server

```bash
yarn start
```

Build assets

```bash
yarn build
```

Run tests

```bash
yarn test
```

Lint

```bash
yarn lint
```

Fix

```bash
yarn lint-fix
```

Publish (build, tests & lint)

```bash
npm run publish
```
